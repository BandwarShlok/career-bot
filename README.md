# Career Compass

Career Compass is a full-stack web application that helps students explore suitable career paths through assessments, counselling plans, and a personalised dashboard.

---

## Project Objective

To provide career guidance by allowing users to:
- Take career assessment tests
- Book counselling plans
- View results and plans in a personalised dashboard

---

## Features

- User Registration and Login
- Secure user-specific data handling
- Career assessment tests
- Counselling plan booking
- Personalised dashboard
- Logout and session management

---

## Technology Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

---

## System Architecture

Frontend communicates with backend using HTTP requests.  
Backend processes data and stores it in MongoDB.

---

## Authentication & Security

- Passwords are encrypted using bcrypt
- Each user accesses only their own data
- Session handling using localStorage
- Protected routes for logged-in users

---

## Dashboard

The dashboard shows:
- Number of tests taken
- Best test score
- Number of counselling plans booked
- Test results and plan history

---
